# OSMNVIM

This repo holds my personal [Neovim](https://neovim.io) config.

Osmnvim was *very* heavily inspired by both [LazyVim](https://github.com/LazyVim/LazyVim) and [NvChad](https://github.com/NvChad/NvChad), as I diverged from using a "distro" config when wanting to lighten up the load on my weaker systems and trim off the fat I didn't need.

This config is ***not*** intended to be a fully personalized project that is "released". This is simply where I store the config I use routinely. 
As such, I will not be able to credit every user I've borrowed/taken snippets from, as this config is an amalgamation of lots of ideas. Biggest credits go to the 'distros' mentioned above.

Naturally, as this is the config for my main laptop, it will also be filled out to fit that system's needs.
Once again, this is ***not*** a "copy the repo and go" config, there will almost certainly be conflicts if you do not use the same applications that I do.
